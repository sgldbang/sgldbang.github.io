# 灵笼雾兰麦朵角色网站项目

## 项目概述

这是一个基于 Jekyll 的 GitHub Pages 网站，专门用于展示《灵笼》第二季中的角色"雾兰麦朵"的个人资料。

### 项目特点
- **主题角色**: 雾兰麦朵（Wulan Maiduo）- 龙骨村的正义监护人和六合枪法大师
- **技术栈**: Jekyll + GitHub Pages + HTML/CSS/JavaScript
- **网站语言**: 中文（主要）+ 英文
- **托管平台**: GitHub Pages (https://sgldbang.github.io)

## 项目结构

```
D:\sgldbang\sgldbang.github.io\
├── _config.yml          # Jekyll 配置文件
├── index.md             # 主页 - 关于雾兰麦朵介绍
├── story.md             # 故事情节页面
├── pet.md               # 宠物查盖页面
├── hobbies.md           # 能力介绍页面
├── _layouts/            # 页面布局模板
├── _includes/           # 可重用组件
├── assets/              # 静态资源
└── images/              # 图片资源
```

## 核心配置

## 网站技术

- **框架**: Jekyll
- **托管**: GitHub Pages
- **主题**: 基于 Minimal Mistakes 的自定义主题
- **样式**: LESS 预处理器
- **脚本**: jQuery + 自定义 JavaScript

## 维护说明

### 内容更新
- 角色信息更新需要同步到相关页面
- 新增页面需要在 _config.yml 中添加导航链接
- 图片资源需要优化后添加到 images 目录
